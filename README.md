# web构建工具，主要包含php, composer，nodejs，glup，webpack等工具

web构建工具，主要类似laravel，Flarum等项目的构建

- 基础镜像：`php-cli:5.6-slim`

## 安装的内容

### 基本工具

- git
- vi
- zip, unzip

### php相关

- php扩展 
  - PDO
  - mcrypt
  - mysqli
  - mbstring
- composer, 使用国内镜像

### nodejs相关

- nodejs 
- npm, 使用国内镜像，取消https
  - glup 
  - webpack
